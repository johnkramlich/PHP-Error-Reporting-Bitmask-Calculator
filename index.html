PHP Error Reporting Bitmask Calculator

When specifiying php error reporting in places such as .htaccess files or apache conf files, it is neccessary to use bitmasked values. This javascript calculator will evaluate a PHP error constant string into it's equivliant bitmasked value. More Info

1 E_ERROR Fatal run-time errors. These indicate errors that can not be recovered from, such as a memory allocation problem. Execution of the script is halted.

4 E_PARSE Compile-time parse errors. Parse errors should only be generated by the parser.
